# CC26 Wall Snippets

Pseudo-code snippets for wall decorations at **Knowledge / CreatorCon**. Playful, dev-flavored code that captures the energy of building, hacking, and community — themed around hackathons, ServiceNow development, and developer culture in general.

Think: vibes over implementation. The kind of code that makes a developer smile and nod.

## Structure

Snippets are organized by wall zone:

| File | Zone |
|------|------|
| [snippets/vibecode.js](snippets/vibecode.js) | Vibe Code zone |
| [snippets/hackathon.js](snippets/hackathon.js) | Hackathon zone |
| [snippets/community.js](snippets/community.js) | Community zone |
